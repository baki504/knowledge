# Visual Studio Code

頻繁に使用するショートカット、追加する設定、プラグインなど

### Mac の修飾キー/シンボル対応表

| 修飾キー              | シンボル |
| --------------------- | -------- |
| command               | ⌘        |
| shift                 | ⇧        |
| option (または alt)   | ⌥        |
| control (または ctrl) | ⌃        |

## 設定

| 設定                             | 項目                           | 設定値 | 備考 |
| -------------------------------- | ------------------------------ | ------ | ---- |
| 別タブでファイルオープン         | workbench.editor.enablePreview | false  |
| ファイル保存時の自動フォーマット | editor.formatOnSave            | true   |
